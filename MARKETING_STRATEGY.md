# 2. Повний каталог можливостей

## 2.1 AI-чат (Flagship Feature)

### In-App AI Chat (повноцінний чат у додатку)

AI Keyboard включає повноцінний AI-чат, порівнянний з ChatGPT, але вбудований прямо в додаток клавіатури.

**Можливості:**
- Вільний текстовий ввід -- запитуйте що завгодно, як у ChatGPT
- **Markdown-форматування** -- таблиці, блоки коду, списки, цитати, заголовки
- **Історія сесій** -- всі розмови зберігаються з автоматично згенерованими назвами. Продовжуйте будь-коли
- **AI-пам'ять** -- асистент запам'ятовує ваше ім'я, професію, вподобання та використовує їх у відповідях
- **Пошук в інтернеті** -- актуальна інформація з мережі через Perplexity з посиланнями на джерела
- **Пошук фото** -- знаходить якісні стокові фото через Pexels з фільтрами та збереженням
- **Промпти в чаті** -- застосовуйте ваші кастомні промпти до відповідей AI як пост-обробку
- **Уточнювальні запитання** -- AI ставить запитання з варіантами відповіді для точніших результатів
- **Глибокі посилання** -- AI створює кнопки для відкриття email, SMS, дзвінків, карт прямо з чату
- **Вибір моделі** -- працює на Gemini 2.5 Flash з автоматичним фолбеком

**Цінність для користувача:** Замінює окремий додаток ChatGPT. Не потрібно платити $20/міс за ChatGPT Plus -- AI Keyboard PRO коштує лише $2.99/міс і включає ВСЕ.

### Інтерактивний чат у клавіатурі

Компактний AI-чат прямо в клавіатурі, доступний у будь-якому додатку.

**Можливості:**
- Діалоги через натискання кнопок -- не потрібно друкувати
- Розумні кнопки дій: вставити текст, відкрити посилання, подзвонити, написати email
- Пошук в інтернеті та пошук фото прямо з клавіатури
- Синхронізація з основним додатком -- історія спільна
- Контекстний аналіз -- AI бачить текст у полі вводу і пропонує релевантні дії

**Цінність для користувача:** AI-асистент завжди під рукою. Не потрібно перемикатися між додатками -- AI працює прямо там, де ви друкуєте.

## 2.6 Пошук в інтернеті

**Технологія:** Perplexity Sonar API

**Можливості:**
- Пошук актуальної інформації в реальному часі
- Цитування джерел з прямими посиланнями
- Гео-оптимізація результатів (регіон визначається автоматично)
- Інтелектуальна фільтрація джерел за якістю
- Повторний запит з суворим фільтром при низькій якості результатів

**Цінність для користувача:** Google та Wikipedia прямо в клавіатурі. Запитайте "скільки населення в Києві" -- і отримайте відповідь з джерелом, не відкриваючи браузер.

**Технологія:** Pexels API (безкоштовна бібліотека якісних фото)

**Можливості:**

- Пагінація (завантаження більше результатів)
- Збереження в Camera Roll одним натисканням

**Цінність для користувача:** Знайдіть ідеальне фото для посту в Instagram чи презентації, не виходячи з чату.

## 2.8 AI-пам'ять

**Технологія:** CoreData з синхронізацією через App Group

**Можливості:**
- AI автоматично запам'ятовує факти про вас (ім'я, професія, вподобання, мова)
- До 50 фактів зберігаються між сесіями
- Факти використовуються для персоналізації всіх відповідей
- Повний контроль: перегляд, видалення окремих фактів, очищення всіх
- Захист від prompt injection (санітизація даних)
- Налаштування в Settings -> AI Memory

**Цінність для користувача:** AI знає вас -- відповіді стають точнішими та релевантнішими з кожною розмовою. Не потрібно щоразу пояснювати контекст.

**Цінність для користувача:** Створіть AI-інструмент для будь-якої задачі. Промпт "Перетвори на поезію" або "Напиши відповідь клієнту в стилі нашої компанії" -- обмеження тільки уява.

## 2.16 AES-GCM шифрування (Cipher)

**Вбудована функція шифрування/дешифрування тексту прямо з клавіатури.**

**Технічні характеристики:**
- **Алгоритми:** AES-GCM (стандарт банківської безпеки) або ChaCha20Poly1305
- **Кодування:** Z85 (компактне) або Base64URL
- **Стиснення:** LZFSE для зменшення розміру
- **Ключ:** HKDF-SHA256 з налаштовуваною сіллю (0/8/16 байт)
- **Автоматичний режим:** Визначає зашифрований текст і автоматично перемикається між шифруванням та дешифруванням
- **Офлайн:** Працює повністю локально, без інтернету

**Як працює:**
1. Напишіть повідомлення
2. Натисніть промпт "Cipher"
3. Введіть пароль
4. Текст шифрується -> вставляється в поле вводу
5. Отримувач з таким самим промптом та паролем натискає "Cipher" -> текст розшифровується

**Цінність для користувача:** Секретне листування прямо в будь-якому месенджері. Telegram, WhatsApp, SMS -- ваші повідомлення зашифровані додатковим рівнем безпеки.
